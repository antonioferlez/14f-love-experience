An interactive romantic web page with a real-time countdown to **February 14th, 2026 (00:00:00)**.

Before the date arrives, the envelope remains locked and the message is blurred.  
Once the countdown ends, the envelope unlocks, opens with animation, and reveals the message with a celebration effect.

---

## Features

- Real-time countdown to February 14th, 2026
- Locked envelope before the target date
- Blurred message while locked
- Animated envelope opening effect
- Automatic unlock when the date is reached
- Heart-shaped fireworks celebration animation
- Minimal romantic aesthetic design
- Fully responsive layout

---

## Technologies Used

- **HTML5**
- **CSS3** (animations, variables, responsive design)
- **Vanilla JavaScript** (time logic, state control, canvas animation)

No backend or database required.
